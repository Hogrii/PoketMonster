## 디스플레이
1. 야생 포켓몬 잡기
2. 포켓몬 확인하기
3. 가방 확인하기
4. 상점 방문하기

## 야생 포켓몬
1. 난수를 발생시킨다(1~151)
    1. 발생된 난수를 키값으로 이용해 해당 번호의 포켓몬을 가져온다
    2. 가져온 포켓몬의 이름을 이용해 포켓몬 객체(다형성)를 생성한다
    3. 해당 포켓몬을 6마리까지 소지할 수 있다
        1. 소지 포켓몬이 6마리가 되면 포켓몬 박스로 전송시킨다
    4. 등장하는 포켓몬 확률 조정
    
## 포켓몬 확인하기
1. 현재 소지하고 다니는 포켓몬(6마리)
2. 포켓몬 박스 확인하기
	1. 현재까지 잡은 포켓몬 출력
3. 도감 확인하기
	1. 현재까지 잡은 포켓몬 마리 수 출력

## 가방 확인하기
1. 몬스터볼 수 확인하기
2. 도구 확인하기
